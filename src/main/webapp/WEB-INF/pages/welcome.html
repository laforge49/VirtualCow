Welcome to Virtual COW

This is a test web site for a new kind of database, Virtual COW. You are welcome to come and kick the tires and take it for a quick drive around the block.

To begin you will need to create a new account, which requires a valid email address. Once you provide your email address you will be sent a token for creating your account. The token is valid for one day only and is tied to your email address.

There is no privacy at this web site. All information is accessible to all users, including information which has been changed or deleted.

Passwords are secured using a seeded secure hash and cookies contain only a login token valid for 3 days. But when you create an account or log in, passwords are passed in the clear.

Finally, there are no backups. Remember, this is just a test site.
